Vraag 1: Wat is de return value van de confirm functie als de gebruiker op een van de buttons klikt?

Antwoord: True bij "ok" en False bij "cancel"

Vraag 2: Wat is de return value van de prompt functie als de gebruiker een tekst intypt en op 'ok' klikt?

Antwoord: Hetgene dat ze hebben ingetypt (string)

Vraag 3: Wat is de return value van de prompt functie als de gebruiker op 'cancel' klikt?

Antwoord: null
